做T原则：
上涨趋势开盘多低开，可开开盘买入，下午2点-2:30卖出。
下跌趋势开盘不要买入做T，等到11点择机而入。

2019.7.9:科创板即将上市，但大盘整体低位呈上升趋势，白酒领涨，贡献大部分大盘指数。(因为国家不断救市)
2019.7.11：不要轻易开新仓（没有思考了分钟以上的），大盘放量下跌不卖票。

入选优质理由：
1.不转不送一般优质。


不入选理由：
1.低于10元的慎重。
2.多头较少且呈下坡路线慎重。
3.财报不好、暴雷股，股价下跌不要立即加仓，可适当减仓。（比亚迪，兴业银行，大族激光就是例子）
4.赚钱卖了的股票不要急着接回来。
5.冒险情况下每只股投资不操过5W。
案例：
1.新城控股董事张猥琐女童，股票断崖式下跌，静待跌停板。

2.大族激光收入跌60%，股价两连跌9%左右，后续继续下跌，后爆出欧洲项目有问题更是下跌。
